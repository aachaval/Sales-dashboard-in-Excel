# 📊 Sales & Shipping Dashboard

## 🧾 Executive Summary
This project focuses on building an automated sales and shipping dashboard using Excel, Power Query, and Power Pivot.

The solution integrates multiple yearly CSV files into a unified data model, enabling dynamic reporting and analysis of sales performance, customer behavior, and shipping efficiency.

New data can be easily incorporated by adding updated CSV files to the designated folder, allowing the dashboard to refresh automatically with minimal manual effort.

---

## 🎯 Business Problem
Many companies manage sales and logistics data across multiple files and periods, making it difficult to:

- Track performance over time  
- Identify top customers and products  
- Analyze shipping costs and delivery efficiency  
- Maintain consistent and reliable reporting  

Manual reporting processes are time-consuming and prone to errors, limiting the ability to make timely, data-driven decisions.

---

## 💼 Use Case

Designed for small to mid-sized businesses managing sales data across multiple periods without a centralized BI system.

---

## ⚙️ Methodology

### Data Sources
- Yearly CSV files containing sales and shipping data  

### Data Processing
- Data loading and transformation using Power Query  
- Data cleaning and standardization across different years  
- Creation of calculated fields and measures  

### Data Modeling
- Relational data model built using Power Pivot  
- Definition of relationships between tables  
- Optimization for efficient querying and analysis  

### Visualization
- Interactive dashboard built in Excel  
- Use of slicers for filtering by year, category, salesperson, and shipping mode  

### Automation
- Scalable structure allowing new yearly data to be added easily  
- Automatic refresh of the dashboard when new CSV files are included

## 📈 Results & Business Insights

The dashboard enables:

- Clear visibility of sales trends over time  
- Identification of top customers and best-performing products  
- Analysis of discount distribution by vendor  
- Evaluation of shipping performance (cost and delivery time)  
- Comparison of performance across salespeople and categories  

### 💡 Business Recommendations

- Focus on high-performing product categories to maximize revenue  
- Optimize discount strategies based on vendor performance  
- Improve shipping efficiency by analyzing delivery times and costs  
- Prioritize key customers contributing most to total sales

## 📂 How to Use

3. The dashboard will automatically update with the new data
